Version 3.0.0
  • Feature: External Extension API. Other locally installed extensions can communicate to this wallpaper service. You would have to grant them access when they initiate a request. You can block or allow such request.
  • Feature: Automatically save options.
  • Feature: New Options User Interface.
  • Feature: Setting wallpaper much faster, uses photos from the cache first if exists otherwise downloads it.
  • Bugfix: First load now shows correct position preference.
